# img2bntx
A command line tool for converting common image formats into Nintendo BNTX textures

```
img2bntx 1.0.0

USAGE:
    img2bntx [OPTIONS] <image> <out>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
        --height <height>
    -n, --name <name>
        --width <width>

ARGS:
    <image>
    <out>
```
